Para poder probar todas las funcionalidades de la aplicación será necesario como mínimo crear un usuario de tipo
"Administrador", con el cual podremos usar la función de crear otros usuarios (Editar Cuentas->Añadir una nueva cuenta).
También se podrán crear los otros tipos de usuarios a mano.


PARA WINDOWS
- Para ejecutar la aplicación Backend bastará con ejecutar el archivo "ejecucionback.bat".
- Para ejecutar la aplicación Frontend bastará con ejecutar el archivo "ejecucionfront.bat".

PARA MAC
Abre la Terminal y navega a la ubicación donde se encuentren los archivos de ejecución. 

Luego, haz los archivos ejecutables con los siguientes comandos:
- (Para el Backend): chmod +x ejecucionback.sh
- (Para el Frontend): chmod +x ejecucionfront.sh

Finalmente, ejecuta los scripts con los siguientes comandos en la Terminal:
- (Para el Backend): ./ejecucionback.sh
- (Para el Frontend): ./ejecucionfront.sh
